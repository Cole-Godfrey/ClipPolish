# Support

## Getting Help

For usage questions and troubleshooting, open a GitHub issue.

## Bug Reports

Use the Bug Report template when behavior is incorrect or regressed.

For hotkey issues, include:
- Whether ClipPolish was launched via `swift run` or from an installed `.app`.
- Accessibility status for ClipPolish in `System Settings -> Privacy & Security -> Accessibility`.
- Whether ClipPolish was quit and relaunched after enabling Accessibility.
- If `tccutil reset Accessibility com.clippolish.app` was run, and whether permission was re-requested after reset.
- The exact in-app message shown (`Shortcut is unavailable` vs `Hotkey blocked by Accessibility permission`).
- Whether manual `Clean Clipboard Text` works while hotkey does not.

## Feature Requests

Use the Feature Request template for new ideas or enhancements.
